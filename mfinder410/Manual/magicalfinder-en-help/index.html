PCs Running Windows

Executing MagicalFinder.exe launches Magical Finder to automatically search for compatible products on the network. Please wait for a moment until products are displayed.

If "Windows protected your PC" is displayed

There are no particular problems. Click [More info], then click [Run anyway].

If "Windows Firewall has blocked some features of this app" is displayed

There are no particular problems. Click [Allow access].

Magical Finder Functions

Change Network Setting

Magical Finder allows you to change the PC name or workgroup, or configure network settings such as IPv4 or IPv6 setting.
The items available for setting in this option vary depending on the device type or system version.
The below screenshots are of PCs running Windows.
When the device is currently connected to the Active Directory domain, you cannot change the network setting.

  1. Click (tap) the device you want to change the network setting for.

  2. Click (tap) [Change Network Setting].

  3. ①Enter the administrator password of the device.
    For the default administrator password, refer to the manual of the target device.
    ②Click (tap) [OK].

    If the password is not specified

    The screen shown above is displayed even if a password is not specified.
    In that case, click (tap) [OK] without entering any alphanumeric value.

  4. Configure the network setting, and click (tap) [Apply].

    5. The following * is displayed only for compatible devices.

    Items displayed next to the icon

    Enter the "PC name" in the upper field and the "WORKGROUP" in the lower field.


    IPv4 Setting

    IP Address Setting
    If the [DHCP] check box is selected, the IP address is automatically obtained.
    Clearing this check box sets the IP address to a fixed value, enabling you to specify "IPv4 Address", "Subnet Mask", and "Default Gateway" shown below.

    IPv4 Address
    Enter the IPv4 address.

    Subnet Mask
    Enter the Subnet Mask.

    Default Gateway
    Enter the Default Gateway.

    DNS Setting*
    You can select one from "Disable", "DHCP ON", and "Static".
    If you select "Static", you can specify "Primary DNS" and "Secondary DNS" shown below.

    Primary DNS*
    Enter the Primary DNS.

    Secondary DNS*
    Enter the Secondary DNS.


    IPv6 Setting

    This item is only displayed for network devices that support the IPv6 setting.

    IP Address Setting
    Select "Disable", "Static", "Stateless Auto IP", or "DHCP ON".
    If you select "Static", you can specify "Link Local Address", "Global Address" and "Default Gateway" shown below.

    Link Local Address
    Displays the Link Local Address.

    Global Address
    Enter the Global Address.

    Default Gateway
    Enter the Default Gateway.

    DNS Setting*
    You can select one from "Disable", "DHCP ON", and "Static".
    If you select "Static", you can specify "Primary DNS" and "Secondary DNS" shown below.

    Primary DNS*
    Enter the Primary DNS.

    Secondary DNS*
    Enter the Secondary DNS.


Magical Finder Functions

Open Setting Website

You can open the setting screen of the device.
The below screenshots are of PCs running Windows.

  1. Click (tap) the desired networked device to be specified.

  2. Click (tap) [Open Setting Website].

The Web browser starts up, and the setting screen opens.For information on the setting procedure, refer to the manual of the device.

The setting screen does not open

If the network groups of the terminal and the device are different, the setting screen may not open.Specify the same network group for the terminal and device, and check whether the setting screen opens.

Magical Finder Functions

Open Remote Desktop

If the device supports remote desktop connection, you can open the desktop screen of the device.
The below screenshots are of PCs running Windows.

  1. Click (tap) the desired networked device to be specified.

  2. Click (tap) ［Open Remote Desktop］.

  3. [Remote Desktop Connection] starts up. Enter the user name and password, and click [OK].

For information on the setting procedure, refer to the manual of the device.

If an access attempt fails

Specify the same network group for the terminal and device, and check whether the setting screen opens.

When You Need Help

The Device Is Not Found

Take the following actions.

- After the [Searching] message disappears, click (tap) the [Reload] button.
After the network setting has been changed, it may take one or more minutes for the updated setting to be displayed due to restarting of the device or other reasons.

- Restart your PC or smartphone and take the appropriate action. If the Wi-Fi connection is used, check that Wi-Fi is set to ON on the terminal.

- Check that the device is turned on.

- Check that the LAN cable is connected.

- Make sure your device and device are in the same network.

* In the case of Wi-Fi router/repeater
Make sure your device is connected to the Wi-Fi router/repeater SSID.

* Network camera/NAS (LAN DISK)
Make sure that the device and the network camera/NAS (LAN DISK) are connected to the same router or hub.

- The device may be blocked by a firewall software function.
Temporarily stop the firewall software function; the target device may be found.If this is the case, there is a method to ignore Magical Finder on the firewall software side.For details, refer to the manual of the firewall software.

- If you try the above but can not find it

* Please try to find it on another device.

* If the terminal can be connected by wire, please try to find it by directly connecting the terminal and device with a LAN cable.
